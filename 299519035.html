Learn to Design and Develop for Virtual Reality

In this online course, we will walk you through the fundamentals of developing an application for virtual reality using the Unity Game Engine and the SteamVR SDK.

Topics covered:

  • install and configure the SteamVR plugin (version 2 or later)
  • design environments for scale in VR
  • implement interaction with virtual props 
  • script movement mechanics and avoid simulator sickness
  • apply optimization techniques to make your application performant
  • recreate game mechanics from popular VR games
  • build a capstone project, incorporating game physics and level design

Prerequisites

Before signing up for this course, you should have prior working knowledge of Unity and C#.  
 

Required Hardware

In order to follow along with the curriculum, you will also need a Mac or PC with a VR head-mounted display that supports six-degrees of freedom. 

Supported VR hardware:

  •  Oculus Rift or Rift S 
  •  HTC Vive or Vive Pro 
  • Valve Index

Note that the Oculus Quest will only be supported with the Stretch Goal.  The course will NOT cover mobile VR devices (Oculus GO, Google Cardboard or Samsung GearVR), which offer limited VR interaction.

The core curriculum will cover the SteamVR SDK, allowing you to reach the widest range of fully immersive VR devices.
